# 3Glo — 3D Modeling Studio Landing Page

A landing page for a 3D modeling and interior design studio.

## Features
- ⏱️ Countdown timer for promotional offer
- 📋 Slide-out navigation menu with smooth scroll
- 🏗️ Services section with tab switching (Interior / Exterior / Animation)
- 🖼️ Portfolio carousel — auto-play every 3s, arrows and dot navigation
- 🧮 Dynamic cost calculator (object type, area, rooms, deadline)
- 📬 Contact form with validation

## Tech Stack
- HTML / CSS / JavaScript
- Webpack 5

## Getting Started

### Installation
npm install

### Development (auto-rebuild on save)
npm run watch

### Production build
npm run build
